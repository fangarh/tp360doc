# Структура проекта



```
plugin folder
|-- package.json      # файл описания модулей, параметров плагина
|-- package-lock.json # файл дополнительных конфигураций пакетов
|-- tsconfig.json     # файл конфигурации Type Script
|-- node_modules      # папка с загружаемыми библиотеками и расширениями NPM
    |-- ...
|-- src               # папка с исходниками
    |-- index.ts      # входной файл приложения
    |-- ...
```
